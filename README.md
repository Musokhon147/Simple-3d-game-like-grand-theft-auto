# Simple 3D Game like Grand Theft Auto

This is a Unity 3D project aimed at creating a simple open-world action game with gameplay mechanics inspired by Grand Theft Auto (GTA).

## Overview

The project serves as a foundational prototype for a 3D action game, featuring:
- **Third-Person Camera Control**: Implemented camera mechanics for smooth player navigation.
- **3D Environment**: A setup for exploring a 3D world (utilizing GameDevHQ assets).
- **Unity Universal Render Pipeline (URP)**: Configured for high-quality graphics.

## Getting Started

### Prerequisites
- Unity 2022.3 LTS or later (Recommended)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Musokhon147/Simple-3d-game-like-grand-theft-auto.git
   ```
2. Open Unity Hub.
3. Click **Add** and select the cloned folder `FInalGME300`.
4. Open the project.
